# Manual de Usuario – Lista de Compras

## Cómo ejecutar la aplicación

1. Abrir la terminal en la carpeta del proyecto:

   cd lista-compras

2. Instalar dependencias:

   bun install

3. Iniciar la aplicación:

   bun start

4. Abrir la app Expo Go en el celular.

5. Escanear el código QR que aparece en la pantalla.

---

## Cómo usar la App

* Escribir un producto y presionar **Agregar** → se añade a la lista
* Tocar un producto → se marca como completado
* Mantener presionado un producto → se elimina

---

## Nota

* No se pueden agregar productos vacíos
