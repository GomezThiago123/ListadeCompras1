# ListadeCompras
🛒 Manual de Usuario – App Lista de Compras
📱 Descripción

La aplicación "Lista de Compras" permite al usuario agregar productos, marcarlos como completados y eliminarlos de manera simple e intuitiva.

⚙️ Requisitos

Antes de ejecutar la aplicación, se debe tener instalado:

Node.js
Bun
Expo Go (en el celular)
🚀 Instalación

Clonar el repositorio desde GitHub:

git clone https://github.com/TU-USUARIO/lista-compras

Ingresar a la carpeta del proyecto:

cd lista-compras

Instalar dependencias:

bun install

▶️ Ejecución de la aplicación

Iniciar el proyecto:

bun start

Se abrirá Expo en el navegador con un código QR.
Abrir la aplicación Expo Go en el celular.
Escanear el código QR.
La aplicación se abrirá automáticamente en el dispositivo.
🧠 Uso de la aplicación
➕ Agregar producto
Escribir el nombre del producto en el campo de texto.
Presionar el botón "Agregar" o la tecla Enter.
✔ Marcar como completado
Tocar un producto de la lista.
El producto se marcará como completado (tachado).
❌ Eliminar producto
Mantener presionado un producto.
El producto se eliminará de la lista.
📌 Consideraciones
No se pueden agregar productos vacíos.
La lista no se guarda al cerrar la aplicación.
No requiere conexión a internet para funcionar.
